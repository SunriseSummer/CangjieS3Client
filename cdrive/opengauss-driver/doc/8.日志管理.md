## 日志管理

仓颉openGauss驱动程序支持使用日志记录来帮助解决在应用程序中使用openGauss驱动程序时的问题。

日志管理主要是通过调用opengauss.slog.Default()的属性和函数来完成，可以执行修改日志级别、把日志输出到文件等操作。

```
import stdx.log.*
import stdx.logger.*
import std.fs.*
import cangjie_tpc::opengauss.slog.*
...

// 日志文件保存到C:\\output\\driver.log
var fl=File("C:\\output\\driver.log", OpenMode.Write)
let log = SimpleLogger(fl)
setDefault(log)
Default().level = LogLevel.TRACE
```
