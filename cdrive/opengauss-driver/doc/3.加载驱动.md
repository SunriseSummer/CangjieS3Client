## 加载驱动

1. 在创建数据库连接之前，需要先加载数据库驱动程序。

2. 可以通过DriverManager.getDriver函数来加载数据库驱动程序。

   1. 函数原型

      ```
      public static func getDriver(driverName: String): Option<Driver>
      ```

   2. 示例

      ```
      let drvopt = DriverManager.getDriver("opengauss")
      var drv: Driver
      // 因为有可能会加载不到驱动，所以需要通过match判断加载驱动是否成功
      match (drvopt) {
          case Some(v) => drv = v
          case None =>
              println("cannot get the opengauss driver")
              return 1
      }
      ```
