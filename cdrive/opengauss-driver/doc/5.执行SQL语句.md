## 第5章 执行SQL语句

### 5.1  SQL执行相关的接口原型

```
func prepareStatement(sql: String): Statement    # 功能：通过传入的 sql 语句，返回一个预执行的 Statement 对象实例
```



### 5.2 执行查询类的SQL语句

应用程序通过执行SQL语句来查询数据库的数据，需要按以下步骤执行：

1. 调用Connection的prepareStatement函数创建语句对象。

   ```
   var statement = conn.prepareStatement("select * from simpletest1 where id=?")
   ```

2. 调用Statement的query函数执行SQL语句，并获取数据。

   ```
   var id1 = "id1"
   statement.set<String>(1, id1)
   var queryResult = statement.query()
   var column1 = ""
   var column2 = ""
   while (queryResult.next()) {
   	column1 = queryResult.get<String>(1)
   	column2 = queryResult.get<String>(2)
   	println("${column1.value}, ${column2.value}");
   }
   statement.close()
   ```

   

### 执行更新类的SQL语句

应用程序通过执行SQL语句来更新数据库的数据，需要按以下步骤执行：

1. 调用Connection的prepareStatement函数创建语句对象。

   ```
   var statement = conn.prepareStatement("update simpletest1 set id=?")
   ```

2. 调用Statement的update函数执行SQL语句。

   ```
   var id1 = "id1"
   statement.set<String>(1, id1)
   var updateResult = statement.update()
   statement.close()
   ```
